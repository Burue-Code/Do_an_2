# Domain Overview – Hệ thống gợi ý phim

Hệ thống gợi ý phim cho người dùng theo thể loại. Backend Spring Boot 3 + Frontend Next.js 14/15 App Router. Auth JWT.

## Domains chính
- **auth**: đăng nhập, đăng ký, JWT
- **user**: profile, preferences
- **movie**: phim, chi tiết, thể loại
- **genre**: thể loại
- **comment**, **rating**, **like**: tương tác
- **watchlist**, **watchhistory**: danh sách xem, lịch sử
- **recommendation**: gợi ý rule-based
- **admin**: quản trị
